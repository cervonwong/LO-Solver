2. Add a `headers` property to the return object, alongside `body`. Conditionally include the header only when `apiKey` is truthy:
   ```typescript
   prepareSendMessagesRequest: ({ messages }) => ({
     body: {
       inputData: {
         rawProblemText:
           (messages[messages.length - 1]?.parts?.[0] as { text?: string } | undefined)
             ?.text ?? '',
         providerMode,
         maxRounds: workflowSettings.maxRounds,
         perspectiveCount: workflowSettings.perspectiveCount,
       },
     },
     ...(apiKey && {
       headers: { 'x-openrouter-key': apiKey },
     }),
   }),
   ```

   ```typescript
     }),
   });
   ```
