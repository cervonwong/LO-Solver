```tsx
function StructuredOutputSection({ data }: { data: Record<string, unknown> }) {
  const [open, setOpen] = useState(false);

  return (
    <div className="border-t border-border-subtle pt-2">
      <RawJsonToggle data={{ input: {}, result: data }}>
        <Collapsible open={open} onOpenChange={setOpen}>
          <CollapsibleTrigger className="hover-hatch-cyan flex w-full items-center gap-2 text-left">
            <span className="text-[10px] font-medium text-muted-foreground uppercase">
              Structured Output
            </span>
            <ChevronIcon open={open} />
          </CollapsibleTrigger>
          <CollapsibleContent className="animate-collapsible mt-1">
            <LabeledList data={data} />
          </CollapsibleContent>
        </Collapsible>
      </RawJsonToggle>
    </div>
  );
}
```

Key details:
- Collapsed by default (`useState(false)`) per CONTEXT.md decision
- The `RawJsonToggle` wraps the collapsible. Pass `{ input: {}, result: data }` so the raw view shows the structured output as "Result:" JSON. (The `input` is empty because structured output is a single object, not input/output pair. The toggle just needs something in `result` to show.)
- Actually, consider creating a simpler toggle wrapper or adjusting: RawJsonToggle currently shows both Input and Result. For structured output, it should only show one JSON block. Create a variant or modify the approach:
  - **Better approach**: Create a small `RawJsonSingleToggle` component alongside `RawJsonToggle` that takes `{ data: Record<string, unknown>; label?: string }` and shows a single JSON block when toggled. OR: pass both input and result as the same data object and adjust.
  - **Simplest approach**: Reuse `RawJsonToggle` but make the Input section conditional — if `data.input` is an empty object `{}`, skip rendering it. Update `RawJsonToggle` to check `Object.keys(data.input).length > 0` before rendering the Input Streamdown block.

Use the simplest approach: update `RawJsonToggle` to skip the Input block when `data.input` is an empty object. This is backward-compatible since no existing caller passes an empty input object. Then use `<RawJsonToggle data={{ input: {}, result: data }}>` for StructuredOutputSection.
