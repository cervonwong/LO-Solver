<objective>
Wire abort signal propagation from the API route through the Mastra workflow to every LLM call, and create the cancel endpoint for frontend use.

Purpose: When a user clicks abort, all in-flight OpenRouter LLM calls must stop within seconds. Currently `useChat`'s `stop()` only closes the client-side stream; server-side LLM calls continue burning API credits. This plan closes that gap by exposing the `Run` object for cancellation and threading the abort signal to every `streamWithRetry`/`generateWithRetry` call.

Output: Backend abort infrastructure — cancel endpoint, signal threading, iteration boundary checks.
</objective>

async function handleWorkflowStream({ mastra, workflowId, params, includeTextStreamParts = true }) {
  const { runId, resourceId, inputData, initialState, resumeData, requestContext, ...rest } = params;
  const workflowObj = mastra.getWorkflowById(workflowId);
  const run = await workflowObj.createRun({ runId, resourceId, ...rest });
  const stream = run.stream({ inputData, initialState, ...rest, requestContext });
  return createUIMessageStream({
    execute: async ({ writer }) => {
      for await (const part of toAISdkV5Stream(stream, { from: "workflow", includeTextStreamParts })) {
        writer.write(part);
      }
    }
  });
}
