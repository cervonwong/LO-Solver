## Actionable Issues

### 1. EVAL-03 Scorer Schema Mismatch (Medium — Integration Bug)

**File:** `src/evals/intermediate-scorers.ts` (lines 98-120)
**Impact:** `scoreRuleQuality` silently returns all-zero scores on every eval run.
**Root cause:** Phase 4 changed the workflow step output from `hypothesisTestLoopSchema` (`{iterationCount, testResults}`) to `questionAnsweringInputSchema` (`{structuredProblem, rules}`). The scorer was not updated.
**Fix:** Update `scoreRuleQuality` to accept the new output shape. The `rules` array is available; pass-rate data would need to be added to the step output or the scorer simplified to report rule count only.

### 2. 8 Unsatisfied Requirements in Phases 5-7

Phases 5 (Verification Loop), 6 (UI Events & Rules Panel), and 7 (Hierarchical Trace & Results) have not been started. All 8 requirements mapped to these phases are unsatisfied.
