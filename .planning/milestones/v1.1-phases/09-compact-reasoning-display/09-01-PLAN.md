```css
/* ============================================
   COMPACT REASONING OVERRIDES
   Applied inside .reasoning-compact wrapper on
   agent reasoning Streamdown content.
   ============================================ */

/* --- Reasoning container --- */
.reasoning-compact {
  max-height: 400px;
  overflow-y: auto;
  background: rgba(255, 255, 255, 0.03);
  padding: 8px;
  border: 1px solid var(--border-subtle);
}

/* Tighter vertical spacing (space-y-2 equivalent instead of default space-y-4) */
.reasoning-compact > div > * + * {
  margin-top: 0.5rem !important;
}

/* Thin scrollbar for reasoning container */
.reasoning-compact {
  scrollbar-width: thin;
  scrollbar-color: rgba(255, 255, 255, 0.2) transparent;
}
.reasoning-compact::-webkit-scrollbar {
  width: 6px;
  height: 6px;
}
.reasoning-compact::-webkit-scrollbar-track {
  background: transparent;
}
.reasoning-compact::-webkit-scrollbar-thumb {
  background: rgba(255, 255, 255, 0.2);
}

/* Word break for long unbroken strings */
.reasoning-compact {
  word-break: break-word;
  overflow-wrap: break-word;
}

/* --- Table overrides inside reasoning --- */
.reasoning-compact [data-streamdown='table-wrapper'] {
  max-height: 200px;
  overflow-y: auto;
  overflow-x: auto;
  scrollbar-width: thin;
  scrollbar-color: rgba(255, 255, 255, 0.2) transparent;
}

.reasoning-compact [data-streamdown='table-wrapper']::-webkit-scrollbar {
  width: 6px;
  height: 6px;
}
.reasoning-compact [data-streamdown='table-wrapper']::-webkit-scrollbar-track {
  background: transparent;
}
.reasoning-compact [data-streamdown='table-wrapper']::-webkit-scrollbar-thumb {
  background: rgba(255, 255, 255, 0.2);
}

.reasoning-compact [data-streamdown='table'] {
  font-size: 10px !important;
  line-height: 1.3 !important;
  border-collapse: collapse !important;
  width: 100% !important;
}

.reasoning-compact [data-streamdown='table-header-cell'],
.reasoning-compact [data-streamdown='table-cell'] {
  padding: 2px 4px !important;
  border: none !important;
  border-bottom: 1px solid var(--border-subtle) !important;
}

.reasoning-compact [data-streamdown='table-header-cell'] {
  font-weight: 600 !important;
  border-bottom-color: var(--border) !important;
}

.reasoning-compact [data-streamdown='table-row']:last-child [data-streamdown='table-cell'] {
  border-bottom: none !important;
}

/* --- Codeblock overrides inside reasoning --- */
.reasoning-compact [data-streamdown='code-block'] {
  max-height: 200px !important;
  overflow: hidden !important;
}

.reasoning-compact [data-streamdown='code-block-body'] {
  max-height: 200px !important;
  overflow-x: auto !important;
  overflow-y: auto !important;
  scrollbar-width: thin;
  scrollbar-color: rgba(255, 255, 255, 0.2) transparent;
}

.reasoning-compact [data-streamdown='code-block-body']::-webkit-scrollbar {
  width: 6px;
  height: 6px;
}
.reasoning-compact [data-streamdown='code-block-body']::-webkit-scrollbar-track {
  background: transparent;
}
.reasoning-compact [data-streamdown='code-block-body']::-webkit-scrollbar-thumb {
  background: rgba(255, 255, 255, 0.2);
}

.reasoning-compact [data-streamdown='code-block'] code {
  font-size: 10px !important;
  line-height: 1.3 !important;
}

/* Prevent any child from expanding beyond reasoning container width */
.reasoning-compact [data-streamdown='table-wrapper'],
.reasoning-compact [data-streamdown='code-block'] {
  min-width: 0 !important;
  max-width: 100% !important;
}
```

Key design choices:
- Background `rgba(255, 255, 255, 0.03)` is very subtle, fits the cyanotype theme — sits between transparent and surface-1 (0.04). Creates a visible-but-not-heavy dimming effect.
- `scrollbar-width: thin` (Firefox) + `::-webkit-scrollbar` 6px (Chrome/Edge) for always-visible thin scrollbar per user decision.
- `border-bottom` only on cells (no `border-left`/`border-right`) for horizontal-only row dividers per user decision.
- `min-width: 0` and `max-width: 100%` on table-wrapper and code-block prevent content from pushing the panel wider.
- All font sizes use 10px to match the user decision of consistent compact data treatment.
