# Core Grammatical Generalizations

1.  **Word Order:**
    *   S O V
    *   T_S O V
    *   T_O S V
    *   S = subject, O = Object, T_S = Topicalised subject, T_O = Topicalised object, V = Verb.
2.  **Noun Structure:** Root – Case – Focus.
    *   **Case:**
        *   nominative = `-nu`
        *   topic = `-V`
    *   **Focus:**
        *   before negative verb = `-V`
        *   focus = `-du`
    *   {a, i, u} + V ⇒ {aa, ee, oo}
3.  **Verb Suffixes:**
    |  | With focus | Without focus |
    | :--- | :--- | :--- |
    | **Present** | `-ju-ru` | `-ju-n` |
    | **Past** | `-ja-ru` | `-ja-n` |
    | **Negative** |  | `-an` |
4.  **Phonological Rule:** {k, g} + j > {c, z}

# Part 1 (Correct the mistakes)

**7. English: As for the decoction, *the merchant* poured it.**
*   **Answer:** `sinjakoo ʔacoodunudu çizaru`

**8. English: As for this thing, it does not pour the oil.**
*   **Answer:** `kuree ʔandaa çigan`

# Part 2 (Translate into English)

**9. Okinawan: `hwahwaa sinjakoo takan`**
*   **English:** As for the mother, she does not cook the decoction.

**10. Okinawan: `ʔandaa kurinu tacan`**
*   **English:** As for the oil, this thing cooked it.

**11. Okinawan: `ʔacoodoo riicidu ncuru`**
*   **English:** As for the merchant, he peels *the lychee*.

# Part 3 (Translate into Okinawan)

**12. English: The person made the decoction.**
*   **Answer:** `Qcunu sinjaku hazan`

**13. English: As for the traditional doctor, he does not stir the poison.**
*   **Answer:** `ʔucinaaʔisjaa dukoo kigan`

**14. English: As for the rice plant, *the snake* stirred it.**
*   **Answer:** `ʔnnee hwiibunudu kizaru`

**15. English: As for the merchant, he does not poke the rice plant.**
*   **Answer:** `ʔacoodoo ʔnnee çiçikan`
