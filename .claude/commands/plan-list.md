# List Implementation Plans

List all implementation plans in the project.

## Process

1. Read `docs/plans/main.md` first to understand overall state
2. List all plan files in `docs/plans/`
3. Show filename, date, and first line (title) of each
4. Indicate status if detectable (in progress, completed, blocked)

## Output Format

```
docs/plans/
├── 2024-01-15-auth-feature.md - Authentication Feature
├── 2024-01-20-dark-mode.md - Dark Mode Support
└── 2024-02-01-offline-sync.md - Offline Sync [IN PROGRESS]
```

## Arguments

$ARGUMENTS - Optional filter (e.g., "auth" to show only auth-related plans)
